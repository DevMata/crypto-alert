## Description

A NestJS and MongoDB RESTful API that handles alerts for crypto prices.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
